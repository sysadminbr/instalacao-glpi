# GLPI Automated Installation Script 
This script install GLPI in an automated way for your convenience.  

## Requirements  

- Ubuntu 24.04LTS  
- Internet Access  
- Root access on the server  

  
## Installation
Download and run the installation script.  
```
curl -s https://raw.githubusercontent.com/sysadminbr/instalacao-glpi/refs/heads/main/glpi11-ubuntu24.sh | bash  
```
